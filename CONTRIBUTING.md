# Contributing

Thanks for your interest in contributing to Awesome Claws.
THis repo is only for independent projects inspired by OpenClaw.
Projects dependent on OpenClaw or in the OpenClaw ecosystem will not be accepted.

## How to contribute

1. Fork the repository.
2. Add or update entries in `README.md`.
3. Keep one project per line using this format:
   - `- **[Project](https://github.com/org/repo)** - Language - Short description.`
4. Ensure links are valid and descriptions are concise.
5. Open a pull request with a clear summary of changes.

## Entry guidelines

- Include open-source projects only.
- Link to the main Github repo, not the project website.
- Prefer active and maintained projects.
- Avoid promotional or exaggerated wording.
- Keep descriptions factual and short.
